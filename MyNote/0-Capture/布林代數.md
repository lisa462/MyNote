布林代數（英語：Boolean algebra）在抽象代數中是指擷取了集合運算和邏輯運算二者的根本性質的一個代數結構¹。具體來說，布林代數處理集合運算交集、聯集、補集；和邏輯運算與、或、非¹。

布林代數的基本定理和定律包括²：
- 單一律：$$A.A = A$$，$$A.1 = A$$，$$A.0 = 0$$，$$A + 0 = A$$，$$A + 1 = 1$$，$$A + A = A$$
- 交換律：$$A.B = B.A$$，$$A + B = B + A$$
- 結合律：$$A + (B + C) = (A + B) + C$$，$$A.(B.C) = (A.B).C$$
- 分配律：$$A.(B + C) = (A.B) + (A.C)$$，$$A + (B.C) = (A + B).(A + C)$$
- 吸收律：$$A + A.B = A$$，$$A + A'.B = A + B$$
- 互補律：$$A + A' = 1$$，$$A.A' = 0$$
- 德·摩根定律：$$\neg (A + B) = \neg A . \neg B$$，$$\neg (A . B) = \neg A + \neg B$$

布林代數在電子工程和電腦科學中同在數理邏輯中一樣有很多實踐應用¹。例如，真值可以在邏輯電路中表示為二進位數或電平，這種相似性同樣擴充到它們，所以布林代數在電子工程和電腦科學中同在數理邏輯中一樣有很多實踐應用¹。

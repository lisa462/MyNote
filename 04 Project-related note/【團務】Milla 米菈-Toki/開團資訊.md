```
本劇本設定於一個自創奇幻世界，靈魂能量滲透於土地與施法之中，拓荒與觀測並行，信仰與學術衝突共存。
靈魂流動、能壓異常與未明現象交錯在荒野之中，而拓荒者的任務，除了開疆拓土，還必須與天災、人禍、古老法則以及「無法言說的東西」對峙。
```
